快速开始

macOS 或 Linux 使用官方安装器，Windows x86_64 使用 PowerShell 安装器。归档、平台支持和源码构建请见安装页。

curl -fsSL https://raw.githubusercontent.com/wjhuang88/talos/main/install/install.sh | sh

首次运行且尚未配置模型时，TUI 会打开模型选择器。CI 或其他非交互环境可使用 talos --no-init -p "summarize this repo"。

配置与凭据

Talos 读取 ~/.talos/config.toml。可使用内联 api_key、环境变量名 api_key_env 或 ${ENV_VAR} 替换。内联密钥会被保存，但在配置展示和诊断表面会被脱敏；请保护该文件。

使用 talos config list、talos config get model 和 talos config set model claude-sonnet-4-20250514 管理配置。配置参考列出了 provider、MCP、dashboard、日志和 reasoning 设置。不要把密钥提交到项目配置中。

Provider、模型与变体

内置默认 provider 包括 Anthropic 和 OpenAI；兼容 provider 可配置协议、base URL、模型和凭据。/model 选择器依次展示最近模型/provider、模型；只有所选模型声明了变体时才显示变体选择。

支持 reasoning 的目录模型可以暴露命名的 reasoning effort。选择的变体会与 provider 和模型一起保存；未知或不支持的变体会安全回退并给出诊断。请使用选择器，而不要自行拼接 provider 请求 JSON。

模式、编辑器与命令

Talos 支持交互式 TUI、inline prompt 和适合脚本的 print 模式。TUI 编辑器按显示宽度折行，包含 CJK 字符。支持增强键盘上报的终端使用 Shift+Enter 换行；Ctrl+J 是可移植的换行后备。普通 Enter 发送请求。

在 TUI 输入 / 打开可用命令菜单。常用入口包括 /model、/connect、/help、配置和会话命令；命令是否可用取决于当前运行时。

工具与权限

Talos 内置文件、目录、搜索、编辑、shell、代码智能、Git、HTTP 和 Web 搜索工具。只读操作可按当前策略执行；写入、删除、Git 写操作、shell、网络动作和 MCP 工具都会通过权限管线。

批准前请检查请求的动作。用户可见的工具范围见能力页；权限边界见安全页。

会话、存储与记忆

本地 SQLite 会话支持历史、搜索、清理、维护、导出与 branch/fork 工作流。记忆固化和 exploration ingestion 是本地功能，不是远程知识服务。请在 TUI 中查看当前构建可用的会话命令。

Skills、MCP 与插件

运行时 Skills 从项目 .talos/skills/、用户 ~/.talos/skills/ 和继承的父项目 skill 目录发现。只激活与当前任务有关的 Skill 内容。

MCP server 支持 stdio、SSE 和 Streamable HTTP 传输，并且仍受权限控制。可通过重复的 --plugin DIRECTORY 显式加载本地只读 WASM 包；远程插件分发、市场、可写插件和自动发现尚未发布。

安全边界

Talos 不是远程多用户服务、浏览器自动化能力、Web 批准路由、自主后台守护进程，也不是稳定的 1.0 SDK。它不会绕过可写工具的权限检查。可选 dashboard 默认为本地只读。

密钥、权限、sandbox 姿态和精确非目标请见安全；已发布/计划/研究的划分请见路线图。

排障

  • 没有模型或凭据：启动 TUI 并使用选择器，或检查 talos config list；密钥应显示为 ***。
  • 终端无法区分 Shift+Enter：使用 Ctrl+J，或在支持增强键盘上报且不经 terminal multiplexer 的环境运行。
  • 安装器或归档问题：在安装页确认平台支持，再使用最新 GitHub Release 产物。
  • 需要更多详情：查看根目录README、执行 talos --help，或在移除密钥后提交 issue。

发布与支持

当前 tag 和 GitHub release 历史见发布页。Talos 仍处于 pre-1.0；依赖 API、命令表面或存储格式前，请按已安装版本验证实际行为。
