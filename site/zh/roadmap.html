路线图

Talos 的已发布、计划中与研究中边界。标记为 已发布 的属于 v0.4.0；标记为 计划中 的已纳入范围；标记为 研究中 的属于探索项，暂无承诺时间线。

如何阅读本页

本路线图仅反映项目公共对外边界。详细工程状态在 docs/BOARD.md、迭代记录和 backlog 中。 本页是简要且保守的公共摘要。若本页公共声明与实际已发布行为冲突， 在页面更新前以已发布行为为准。

已在 v0.4.0 发布

以下内容属于当前已发布 release。详见 能力页面和 安全页面。

  • 已发布 TUI、inline 和 print 执行模式。
  • 已发布 按 ADR-023 执行密钥脱敏的本地 provider 配置。
  • 已发布 带权限门控的内置编码工具（文件、搜索、代码智能、Git、 http_request、web_search、 bash）。
  • 已发布 会话存储、搜索、清理、维护、记忆固化与探索库导入。
  • 已发布 来自 .talos/skills/、 ~/.talos/skills/ 和继承父级 .talos/skills/ 的运行时 Skills。
  • 已发布 MCP stdio 工具与 JSON-RPC 基础设施。
  • 已发布 TUI 模式下的只读 loopback dashboard，启动时提示本地 URL 和 token。
  • 已发布 初始 talos-runtime 可嵌入 facade （pre-1.0；ADR-024）。

计划中

以下事项已纳入范围但尚未发布。下述顺序表示优先序偏好，不代表发布时间承诺。

  • 计划中 将 cargo install talos-cli --bin talos 作为已验证的 Cargo 安装路径。当前 --path 源码安装可用； crates.io 发布仍受 发布矩阵 约束，并且需要维护者显式批准。
  • 计划中 为 talos-runtime facade 提供稳定 1.0 SDK 保证。 该项受 REL-002 门控，要求 Talos 作为主运行时完成 100% 自举开发并提供记录证据。 当前正在推进 四个月加固计划 以产出前置条件和证据。
  • 计划中 在当前 http_request / web_search 基础之上，提供有界的 PDF/Office 文档提取与保存工作流（见 WEBFETCH-001）。
  • 计划中 在当前 ADR-026 基础之上，扩展多资源工具权限配置能力。

研究中

以下事项属于探索阶段。它们没有承诺时间线，也不会被描述为已发布。 其中一些作为独立产品差异化工作跟踪，可能不会进入 Talos 核心运行时。

  • 研究中 ~/.agents/skills/ dotagents 共享目录发现。 跟踪项： AGENT-002。
  • 研究中 只读 dashboard MVP 之外的 Web 控制面扩展。 独立跟踪项： WEB-001。 远程访问、浏览器自动化、Web 批准和 Web 写操作/动作路由尚未发布， 也不是安全边界。
  • 研究中 WASM plugin runtime 与插件市场。相关架构决策 已接受（ADR-027/028/029/030）， 但运行时实现仍受聚焦的 wasmtime 依赖和安全审查门控。 跟踪项： PLUGIN-001。
  • 研究中 远程或 P2P 会话控制。跟踪项： REMOTE-001。 远程会话控制面在任何原型工作前都需要 ADR 门控。
  • 研究中 本地微模型决策层（设备端小模型用于提示）。 跟踪项： MODEL-002。

v0.4.0 非目标

以下为明确非目标，提前列出以避免用户通过试错获得这些结论。

  • Talos 不是远程多用户服务。 没有远程控制平面、托管会话路由器，也没有远程批准 UI。
  • Talos 不是 marketplace runtime。 WASM plugin runtime 和插件市场仅处于研究阶段。
  • Talos 不是远程 Web 控制平面或浏览器自动化产品。 已发布 dashboard 仅限本地、只读、loopback-only、token-gated；更广泛的 Web 控制仍是独立研究轨道（WEB-001）。
  • Talos 不是自主后台守护进程。 它以交互式或 print 模式运行，不会在后台等待触发。
  • Talos 不会自动提交、自动发布或自动合并。 Git 写操作只会通过显式工具或用户动作发生。
