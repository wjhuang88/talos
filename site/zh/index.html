v0.2.0 — pre-1.0 发布线

一个安全优先的本地编码 Agent，使用 Rust 编写。

Talos 是一个 Rust 原生本地编码 Agent，面向希望在自己机器上审查、扩展和运行 Agent 运行时的开发者。它提供交互式 TUI、显式权限、运行时 Skills、 MCP/RPC 集成和可嵌入运行时 facade — 同时默认核心保持本地且可审计。

核心亮点

默认本地优先

支持交互式 TUI、inline 模式和用于脚本/冒烟测试的 print 模式。 启动无需依赖远程服务。

显式权限边界

文件写入、删除、Git 写操作、shell 执行、网络动作和 MCP 工具 全部通过权限管线。

Rust 原生核心

采用以 workspace 为中心的 crates，尽量减少运行时假设。 运行 Talos 不需要 Node.js 或 Python 运行时。

可嵌入运行时 facade

talos-runtime crate 让 Rust 项目可以构建 安全的进程内 Agent 运行时，而无需依赖 Talos CLI/TUI crates。

持久会话与记忆

提供 SQLite 支撑的会话历史、搜索、分支/分叉支持、 语义记忆固化和保留策略预览。

可扩展能力面

MCP stdio 工具、hooks、JSON-RPC 和项目治理状态 已实现；plugin/WASM 和浏览器控制面仍处于研究阶段。

当前发布边界

已在 v0.2.0 发布 TUI、inline 与 print 执行模式；本地 provider 配置与密钥脱敏； 带权限门控的内置编码工具；会话存储、搜索、清理、维护、记忆固化 与探索库导入；来自 .talos/skills/、 ~/.talos/skills/ 和继承父级 .talos/skills/ 的运行时 Skills；MCP stdio 工具与 JSON-RPC 基础设施；以及初始 talos-runtime 可嵌入 facade。

尚未发布 可嵌入运行时 facade 的稳定 1.0 SDK 保证、 ~/.agents/skills/ dotagents 共享目录发现、 嵌入式浏览器或 Web 控制面、WASM plugin runtime 与插件市场、 超出当前 web/fetch 基础能力的 PDF/Office 文档解析、以及远程或 P2P 会话控制。完整的已发布 / 计划中 / 研究中划分请见 路线图。
